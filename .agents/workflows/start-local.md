---
description: Come accendere il sito in locale
---

1. Apri **Docker Desktop** sul tuo computer.
// turbo
2. Apri un terminale nella cartella `backend` del progetto.
// turbo
3. Esegui il comando `docker-compose up -d`.
4. Una volta che i container sono avviati, apri il browser all'indirizzo `http://localhost`.
